# 📊 Sample Superstore Dashboard - Power BI

## 📌 Project Description

This Power BI dashboard analyzes the **Sample Superstore dataset** to provide insights into sales, profit, customer segments, and product performance. The dashboard helps identify business trends, top-performing products, and areas that need improvement for better decision-making.

## 🎯 Key Features

* Total Sales, Profit, Orders, and Quantity KPIs
* Sales trend analysis
* Region and state-wise performance
* Category and sub-category analysis
* Customer segment insights
* Top and bottom performing products
* Interactive filters and slicers

## 🛠️ Tools Used

* Power BI
* Power Query
* DAX

## 💡 Insights

* Identify profitable products and regions.
* Analyze the impact of discounts on profit.
* Compare customer segment performance.
* Track overall business growth.
